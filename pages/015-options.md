# The Options

- [ ] `src/components/`  
      Flat inside one app. Files by type.

- [ ] `apps/catalog` + `apps/checkout` + `packages/ui`  
      Microfrontends. Many apps. Shared UI. Independent deploys.

- [ ] `src/module/catalog/`  
      Modular monolith. One app. Folders by feature.
